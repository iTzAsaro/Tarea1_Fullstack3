# Tarea1_Fullstack3

Proyecto con estructura escalable:

- `frontend/`: aplicación React + TypeScript con Atomic Design (`atomos/`, `moleculas/`, `organismos/`, `plantillas/`, `paginas/`).

## Requisitos

- Node.js LTS
- npm (workspaces)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev:frontend
```

## Variables de entorno

- `frontend/.env.example` (Vite): copiar a `frontend/.env` y ajustar valores.

## Calidad de código

```bash
npm run lint
npm run format
npm run typecheck
```

## Tests

```bash
npm test
```

## Notas

- El archivo `e-commers.html` queda como referencia del prototipo original.
